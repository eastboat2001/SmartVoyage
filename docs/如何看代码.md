**最重要的复习方法**
每读完一轮，都强迫自己回答这 5 个问题：
1. 这个模块的职责是什么
2. 输入输出是什么
3. 它依赖谁
4. 为什么这么设计，不是另一种
5. 我怎么测试它

如果你答不上，就说明还没读懂。
